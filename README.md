# slimane-cli
Command line interface for Slimane(https://github.com/noppoMan/Slimane)


## Installation
```sh
npm i -g slimane-cli
```

## Generate a new Slimane app
```sh
slimane new MySLimaneApp
```

## Build
```sh
cd MySLimaneApp
slimane build
```

## Run
```sh
slimane run
# The server is listening at 0.0.0.0:3000
```
